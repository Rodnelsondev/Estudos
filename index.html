o que é HTML ?
html é a linguagem na qual a maioria dos sites são escritos.
O html é usado para criar páginas e torná-las funcionais.
Significa: Hyper Text Markup Language
Em português: Linguagem de marcação de hipertexto

o que são tags e atributos ?
Tags e atributos são a base do html.
Eles trabalham juntos, mas desempenham funções diferentes.

o que são tags?
As tags são usadas para marcar o início de um elemento HTML e geralmente são colocadas entre colchetes angulares.
A maioria das tags deve ser aberta <h1> e fechada </h1>.para funcionar.

o que são atributos HTML?

Os atributos contêm informaçôes adicionais. Os atributos assumem a forma de uma tag de abertura e informaçôes adicionais são colocadas dentro.

um exemplo de atributo é:
< img src="meucachorro.png" alt="a foto do meu cachorro" >

nesse caso,a fonte da imagem "src" e o texto alternativo "alt" são atributos da <img> tag.

Construção basica de uma página html

Essas tags devem ser colocadas uma abaixo da outra no topo de cada página html.
